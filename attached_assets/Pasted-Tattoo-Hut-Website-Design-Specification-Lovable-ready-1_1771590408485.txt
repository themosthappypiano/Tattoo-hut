4. Layout Guide (by page)
Shared layout rules

Max width: 1100–1200px content container

Sections: generous spacing (72–96px vertical on desktop, 48–64px mobile)

Use clear headings + short paragraphs + cards for scannability

Every page should have a visible CTA within the first viewport (above the fold)

Home layout

Hero (split layout) → Proof chips row → “Why us” cards → Services (two feature panels) → Artists strip → Gallery grid preview → Reviews block → FAQ teaser (3 Qs) → CTA banner → Footer

Sticky subnav or tabs (Tattoos / Piercings) → each section has: intro + price note + steps + CTA + gallery strip

5. Style Guide
Brand vibe

Neon-lit neighborhood gem: warm, creative, unpretentious, proudly diverse.

Color palette

Primary background: #111111

Surface cards: #1A1A1A

Neon accent: #FF2200

Glow: rgba(255, 34, 0, 0.35)

Chrome silver: #C0C0C0

Text primary: #FFFFFF

Text muted: #CFCFCF

Typography

Headlines / accents: Script font for select headings (Pacifico / Playlist Script vibe)

Body: Inter / DM Sans / Manrope (clean, modern sans)

Suggested scale:

H1 52 / 46 / 38 (desktop/tablet/mobile)

H2 36 / 32 / 28

Body 18 / 16 / 16

Buttons 16–18 semi-bold

UI components

Primary button: Neon red fill (#FF2200), soft glow, rounded 16px

Secondary button: Transparent, 1px silver border, red glow on hover

Cards: subtle border (silver @ 20% opacity), slight shadow, hover glow

Icons: simple line icons (Lucide style), white/silver with red hover

Dividers: faint “neon tube” horizontal lines (gradient from transparent → red glow → transparent)

Imagery

Dark, warm photography: studio interior, clean stations, artist-at-work, friendly candid moments

Avoid overly “hard/metal” tattoo tropes; keep it welcoming and modern

6. Additional Instructions (logic + behavior)
Booking embed behavior

Book button always routes to /book

If using Booksy: embed widget or deep-link to Booksy booking flow

If using Square: embed appointment widget or “Book now” link + preserve UTMs

Trust & proof

Show “5.0★ • 140+ reviews” consistently (home hero + book page + footer)

Include a “Reviews” section that links out to Google reviews (or embed if allowed)

Payment copy (until confirmed)

Use neutral copy: “Multiple payment options accepted — call to confirm what’s best today.”

Do not list Cash App/Venmo/Card until confirmed

Accessibility

Ensure neon red contrast is AA on dark backgrounds (use white text, reserve red for accents/buttons)

Provide focus states and keyboard navigation for filters/lightbox

Product vision:
Tattoo Hut is a Houston tattoo and piercing shop delivering custom work in a spotless, inclusive, welcoming environment with fair pricing. The website’s job is to convert visitors into bookings, calls, and walk-ins with minimal friction.

Primary audience personas:

Spontaneous walk-in client — wants quick info, price confidence, directions, and a “book now” option

First-timer / nervous client — wants reassurance (cleanliness, warmth, inclusivity, FAQs, aftercare)

Artist-loyal follower — comes for Wyskii/Caelan, wants portfolio + direct booking

Budget-conscious client — needs transparent “starting at $50” and quick booking

Key features:

Embedded online booking (Booksy or Square)

Sticky mobile CTAs (Book/Call/Directions)

Artist profiles + portfolios

Filterable gallery + lightbox

FAQ + trust signals (5.0★ / 140+)

Contact page with map and parking note

Tone of voice:
Warm, confident, inclusive, no-ego. Short, punchy sentences. Emphasize comfort and cleanliness.
